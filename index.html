Flexbox Row

Use these three properties to create a Flexbox row layout

.row{
    display:flex;
    flex-direction:row;
    flex-wrap:wrap;
}

Flexbox Column

Use this to create a Flexbox column layout

.column{
    display:flex;
    flex-direction:column;
}

CSS Grid Layout

Build a 12-column layout using CSS Grid

.grid{
    display:grid;
    width:100%;
    grid-template-columns:repeat(12,1fr);
}

Linear Gradients

This will create a background linear Gradients from top to bottom. 

.linear-gradient-background{
    background-image:linear-gradient(#6b4775,#e867ed);
}

Box Transition GLow

Use transition and box shadows to glow on hover.

.code-card  .card-header{
    border-radius:8px;
    transition:all 0.5s ease-in-out;
}

.code-card:hover, 
.code-card:hover .card-header{
    box-shadow:inset 0px 0px 8px (#6b4775,#e867ed);
}

Overlay Card with Title

Use position properties and negative margines to raise elements higher than their natural starting point. 

.card-header{
